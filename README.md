# Cleaning Service Revenue Analysis

![Google Sheets](https://img.shields.io/badge/Tool-Google%20Sheets-brightgreen) ![Status](https://img.shields.io/badge/Status-Complete-blue)

## Overview
This project analyzes revenue data for a residential cleaning service. Using Google Sheets, I organized and explored customer transactions to uncover service trends, total revenue, and payment patterns — skills directly applicable to business analytics roles.

## Data
Each row represents one service transaction. Columns used:
The dataset is included in this repository as `cleaning-service-revenue-analysis.xlsx`.

| Column | Description |
|--------|-------------|
| Date | Date the service was performed |
| Customer | Client name or ID |
| Service Type | Deep Clean, Standard Clean, etc. |
| Amount | Revenue from that transaction |
| Payment Status | Paid, Pending, or Overdue |
| Notes | Any relevant context |

## Key Insights

- **Deep Clean vs Standard Cleaning revenue** — compared total revenue by service type to identify which generates the most income
- **Total revenue generated** — summed all transactions to get overall business income
- **Payment tracking** — flagged unpaid invoices to monitor outstanding cash flow

## Tools Used
- **Google Sheets** — data entry, SUM/SUMIF formulas, filtering, pivot tables
